# AGENTS.md

## Project Goal

Build a small, explainable AI interview learning backend using FastAPI.

## Working Rules

1. Read relevant files before proposing changes.
2. Provide a short implementation plan before editing.
3. Change only files required by the current task.
4. Do not introduce a framework unless it clearly reduces complexity.
5. Never place secrets or real API keys in source files.
6. Add or update tests for core behavior.
7. Run the relevant tests after changes.
8. Summarize changed files, test results, and remaining risks.
9. Do not rewrite the whole repository for a local feature.
10. Prefer understandable code over clever abstractions.
